Installing
----------

You will need Python 3 and Pygame.

There are binaries for Pygame for Python 3:

* https://bitbucket.org/pygame/pygame/downloads - Windows
* https://launchpad.net/~thopiekar/+archive/ubuntu/pygame - Ubuntu
